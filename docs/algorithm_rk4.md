# 🧮 Алгоритм Рунге-Кутти 4-го порядку

## 📌 Призначення

Метод використовується для чисельного розв'язання диференціальних рівнянь.


## ⚙️ Ідея методу

Використовуються 4 коефіцієнти:

- k1
- k2
- k3
- k4


## 📐 Формула

y(n+1) = y(n) + (h/6)*(k1 + 2*k2 + 2*k3 + k4)


## ✅ Переваги

- точність 
- стабільність 
- простота

## 🚀 У проєкті

Метод використовується для:

- координат 
- швидкостей
