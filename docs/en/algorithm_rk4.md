# 🧮 Runge-Kutta 4th Order Algorithm

## 📌 Purpose

The method is used for the numerical solution of differential equations.

## ⚙️ Idea of the Method

4 coefficients are used:

- k1
- k2
- k3
- k4

## 📐 Formula

y(n+1) = y(n) + (h/6)*(k1 + 2*k2 + 2*k3 + k4)

## ✅ Advantages

- accuracy
- stability
- simplicity

## 🚀 In the Project

The method is used for:

- coordinates
- velocities
