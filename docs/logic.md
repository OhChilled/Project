# 🧠 Логіка роботи програми

## 🎯 Призначення

Програма моделює рух космічного апарата та перевіряє можливість стикування 🚀

## 🔄 Етапи роботи

1. Введення параметрів (`ax`, `ay`)
2. Ініціалізація координат
3. Розрахунок нового стану (RK4)
4. Обчислення відстані
5. Перевірка стикування

## 📏 Умова стикування

Стикування відбувається, якщо:

```text
R <= Rcrit
```

## 📌 Результат

Програма виводить:

✅ стикування виконано
❌ стикування не виконано
