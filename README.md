# Project

## Опис проєкту
Опис проєкту

Даний проєкт є бакалаврською роботою, що реалізує симуляційну модель.

Проєкт написаний мовою C++ та використовує CMake для збірки.

## Структура проєкту

* `cosmos2025.cpp` — основний файл програми
* `library.cpp`, `library.h` — логіка проєкту
* `tests/` — модульні тести
* `CMakeLists.txt` — збірка
* `.github/workflows/` — CI/CD

## Збірка

mkdir build
cd build
cmake ..
make
